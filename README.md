1. Откройте терминал
2. В корне папке в терминале выполните команду 
```
docker image build . --tag=my-nginx
```
3. После завершения процесса создания образа выполните команду для запуска контейнера
```
docker run -d -p 7887:80 my-nginx
```
4. После запуска контейнера выполните GET запрос на адрес localhost порт 7887
```
curl loclahost:7887/
```
